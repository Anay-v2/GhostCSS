Button Groups

GhostCSS has great looking button groups that can be customized. Let's take a look at them:

<div class="btn-group">
<button class="btn btn-red">foo</button>
<button class="btn btn-outline-red">bar</button>
<button class="btn btn-red">baz</button>
</div>

Sizes

GhostCSS automatically changes the sizes of other buttons if one button's size is changed:

<div class="btn-group">
<button class="btn btn-red">foo</button>
<button class="btn-lg btn-outline-red">bar</button>
<button class="btn btn-red">baz</button>
</div>
