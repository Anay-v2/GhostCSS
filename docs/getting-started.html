Getting started

This page will teach you how to get started with ghostCSS.

Installation

This part will teach you how to add GhostCSS in your project

To use GhostCSS you need to have basic knowledge of HTML and Nodejs. The first step is to initialize your project. type this in the terminal:

$ npm init vite my-ghost-project

Now node will ask you what you want to base it on. select none. And if you want, select the TypeScript version. Now open the folder that has been created and then type this command

$ npm install -D postcss tailwindcss autoprefixer

Now wait for them to install and type this command:

$ npx tailwindcss init -p

This is not a typo. npx is a command built within npm. This will create a tailwindcss and postcss config files. So now in tailwind.config.js delete everything and paste this:

module.exports = {
mode: 'jit',
purge: ['./index.html', './*.{html,htm}'],
darkMode: 'media', // or 'media' or 'class'
theme: {
extend: {
zIndex: {
'top': '10000'
},
spacing: {
'128': '32rem',
'144': '36rem',
},
screens: {
'portrait': { 'raw': '(orientation: portrait)' },
'm2xl': { 'max': '1535px' },
'mxl': { 'max': '1279px' },
'mlg': { 'max': '1023px' },
'mmd': { 'max': '767px' },
'msm': { 'max': '639px' },
},
borderWidth: {
'1': '1px'
}
},
},
variants: {},
plugins: [],
}

If you want you can change darkMode from 'media' to 'class' or even false. This affects the dark mode.

Now delete the default css file and make a new one called 'index.css' and paste this in it. You might want to turn off any formatters before this or else some functionality of the components may not work.

Now you can get started with your website! Just add this in the head tag:

<link rel="stylesheet" href="./index.css">

And you're ready to go!
